Audio Compression

The goal of this project is to explore the uses of audio compression. The purpose of audio compression is to appproximate an original sound with a file that is smaller in size so that it is easier to store and transfer. Part 5

For the last part, I decided to compress the last 10 seconds from a concert band piece called "Pageant" by Vincent Persichetti. The whole song can be heard here. The recording was done by the North Texas Wind Symphony and is very well done. The reason I chose this piece is for the very end, which is about the last 15 seconds of the recording. The ending is quite dramatic with a brass chorale, and it ends with an interesting chord.

To listen to the last few seconds, see below. The ending is a little avante garde for some because the last chord is all chromatic, which means it is quite dissonant. Despite this, there is actually an emphasis on a B-flat major chord. One can hear it if they listen closely. The trumpet is that is quite audible is playing a high concert B-flat.
